# 儿童老电视播放器

基于 Vue3 + Ant Design Vue 的儿童视频播放器，外观是复古老电视风格，支持 10 个频道切换。

## 功能

- 老电视 UI + 乐高风格频道按钮（1~10 台）
- 每个频道可单独配置视频链接
- 家长模式：先通过 100 以内加减法验证，再进入设置
- 每个频道分别记录播放进度（切台回来继续播放）
- 本地存储频道链接和播放记录（localStorage）

## 本地运行

```bash
npm install
npm run dev
```
